¿Qué son las Variables CSS?

¡Organiza y Reutiliza tus Estilos!

Las variables CSS (también conocidas como propiedades personalizadas) te permiten definir valores que puedes reutilizar en todo tu CSS. Esto te ayudará a mantener la consistencia en tu diseño y a realizar cambios globales de forma muy sencilla.

Al cambiar un solo valor en la definición de la variable en `:root`, todos los elementos que la utilizan se actualizarán automáticamente. Esto permite crear temas o realizar ajustes de diseño de manera mucho más ágil.

¡Demostración Clave! Una Variable, Múltiples Usos

**Observa cómo el título `h2` de esta sección y el botón de abajo comparten el mismo color principal.** Esto se logra utilizando la variable CSS `--color-primary` en dos propiedades diferentes: el `color` del texto para el `h2` y el `background-color` para el botón. Si cambias el valor de `--color-primary` en `:root`, ¡ambos elementos se actualizarán automáticamente!
